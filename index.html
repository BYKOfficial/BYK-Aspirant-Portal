Google Forms — Timer Quiz
Create MCQ tests with auto-scoring, answer keys, and result sheets. Embed directly in any page.
  • → forms.google.com → New Blank Form
  • → Settings → Make it a Quiz → Auto-grade ON
  • → Send → Embed () → Copy iframe code
  • → Paste in your site's HTML section
💡 Pro Tip: Use Google Apps Script to auto-send results to Sheets and build a live leaderboard — completely free!
⏱
Jotform — Countdown Timer Quiz
Jotform's free plan supports built-in countdown timer for quizzes. Best alternative to paid TestMaker platforms.
  • → jotform.com → Create Form → Quiz Mode
  • → Add Timer widget from Widget Gallery
  • → Set 'Auto-submit on timer expiry'
  • → Embed → Get Code → Paste in site
💡 Pro Tip: Jotform's conditional logic creates adaptive quiz flow — shows different questions based on previous answers, totally free!
📊
Google Sheets — Progress Dashboard
Connect Forms to Sheets for auto-logging. Use Looker Studio (free) to build visual dashboard embeddable on site.
  • → Form responses auto-sync to Sheets
  • → lookerstudio.google.com → Create Report
  • → Connect Sheet as data source
  • → File → Embed Report → Copy iframe
💡 Pro Tip: Use IMPORTRANGE() to combine data from multiple form sheets into one master leaderboard dashboard!
📄
Google Drive PDF Viewer
Host PDFs on Drive and embed them as in-site viewers. Students read directly in browser — no downloads needed.
  • → Upload PDF to Google Drive
  • → Share → Anyone with link (Viewer)
  • → Copy URL → replace /view with /preview
  • → Embed iframe with width="100%"
💡 Pro Tip: Add ?toolbar=0&navpanes=0 to the URL to hide Google's UI for a cleaner embedded reading experience!
💬
Disqus / Telegram — Community
Add peer-to-peer discussion on each study material page. Disqus is free, no backend, 5-minute setup.
  • → disqus.com → Get Started → Create Site
  • → Copy Universal Embed Code
  • → Paste in page footer section
  • → Create free Telegram group for live chat
💡 Pro Tip: Embed a Telegram widget on your site as real-time chat. Telegram communities are more active than Disqus for Indian exam students!

Netlify / GitHub Pages — Hosting
Host this complete site 100% free with SSL, CDN, and custom domain. Upload and go live in under 2 minutes.

  • → OR: github.com → New Repo → Upload files
  • → Settings → Pages → Source: main → Save
