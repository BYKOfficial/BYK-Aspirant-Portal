Zero-Cost Power Stack

Build Everything Free. No Compromise.

Exact tools and embed workflows used to build this site. Step-by-step for every widget.

📋
Google Forms — Timer Quiz
Create MCQ tests with auto-scoring, answer keys, and result sheets. Embed directly into any website page using the embed code.
  • → forms.google.com → New Blank Form
  • → Add MCQ questions with correct answers
  • → Settings → Make it a Quiz → Enable auto-grading
  • → Send → Embed () → Copy iframe code
  • → Paste in your site's HTML section
💡 Pro Tip: Use Google Apps Script to auto-send results to a Google Sheet and build a live leaderboard for free!
⏱
Jotform — Countdown Timer Quiz
Jotform's free plan supports a built-in countdown timer for quizzes. Best alternative to paid platforms like TestMaker.
  • → jotform.com → Create Form → Quiz Mode
  • → Add Timer widget from Widget Gallery
  • → Set duration (e.g., 120 mins for UPSC mock)
  • → Set 'Auto-submit on timer expiry'
  • → Embed → Get Embed Code → Paste in site
💡 Pro Tip: Jotform's conditional logic lets you show different follow-up questions based on previous answers — creating an adaptive quiz flow for free!
📊
Google Sheets — Progress Dashboard
Connect Google Forms to Sheets for auto-logging. Use Data Studio (Looker Studio) to build a visual progress dashboard embeddable on site.
  • → Forms responses auto-sync to Google Sheets
  • → lookerstudio.google.com → Create Report
  • → Connect your Sheet as data source
  • → Add charts, scorecards, bar graphs
  • → File → Embed Report → Copy iframe → Paste
💡 Pro Tip: Use IMPORTRANGE() in Sheets to combine data from multiple form sheets into one master dashboard for class-wide tracking!
📄
Google Drive PDF Viewer
Host PDFs on Google Drive and embed them as in-site viewers. No downloads needed — students read directly in browser.
  • → Upload PDF to Google Drive
  • → Right Click → Share → Anyone with link (Viewer)
  • → Open → Copy URL → replace /view with /preview
  • → Use in iframe: src="PASTE_URL"
  • → Add width="100%" height="700px"
💡 Pro Tip: Use ?toolbar=0&navpanes=0 parameters in the URL to hide Google's UI for a cleaner embedded reading experience!
💬
Disqus / Giscus — Community Chat
Add peer-to-peer discussion threads on each study material page. Disqus is free, no backend needed, 5-minute setup.
  • → disqus.com → Get Started → Create Site
  • → Copy the Universal Embed Code
  • → Paste in your page's footer section
  • → Students can comment, upvote, reply
  • → For GitHub users: giscus.app is even cleaner
💡 Pro Tip: Create a dedicated Telegram group (free) and embed a Telegram widget on your site as a real-time chat alternative to Disqus!
🚀
Hosting: Netlify / Vercel / GitHub Pages
Host this entire website 100% free with SSL, CDN, and custom domain support. Upload files and go live in under 2 minutes.
  • → netlify.com → New Site → Drag & Drop HTML folder
  • → OR: github.com → New Repo → Upload files
  • → Settings → Pages → Source: main → Save
  • → Site live at username.github.io/repo-name
  • → Add free .in domain via Freenom or buy ₹99/yr
💡 Pro Tip: Use Netlify Forms (free) for lead capture. No backend required — form submissions go directly to your email and Netlify dashboard!
